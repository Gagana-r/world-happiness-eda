# 🌍 World Happiness Report - Exploratory Data Analysis

An exploratory data analysis (EDA) of the World Happiness Report dataset (2015-2019) using Python, Pandas, Matplotlib, and Seaborn.

## 🔗 Links
- **GitHub:** https://github.com/Gagana-r/world-happiness-eda
- **Kaggle Notebook:** https://www.kaggle.com/code/gaganasri22/worldhappinesseda

## 📌 About
This project analyzes global happiness scores across 156+ countries from 2015 to 2019. It explores which factors most strongly influence happiness and how scores have changed over time.

## 🛠️ Tech Stack
| Tool | Usage |
|---|---|
| Python | Programming language |
| Pandas | Data manipulation |
| NumPy | Numerical computing |
| Matplotlib | Data visualization |
| Seaborn | Statistical visualization |
| Kaggle Notebooks | Development environment |

## ✨ Analysis Performed
1. **Top & Bottom 10 Countries** — Bar charts of happiest and least happy countries
2. **Correlation Heatmap** — Relationship between happiness factors
3. **GDP vs Happiness** — Scatter plot showing economic impact
4. **Trend Analysis** — Line chart of top 5 countries (2015-2019)
5. **Key Insights Summary** — Statistical findings

## 🔑 Key Findings
- 🏆 **Finland** is the happiest country in 2019 (Score: 7.769)
- 😔 **South Sudan** is the least happy (Score: 2.853)
- 💰 **GDP per capita** has the strongest correlation with happiness (0.794)
- ❤️ **Healthy life expectancy** is second strongest (0.780)
- 🤝 **Social support** closely follows (0.777)
- 😇 **Generosity** has the weakest correlation (0.076)
- 📊 Average happiness score in 2019: **5.407**
